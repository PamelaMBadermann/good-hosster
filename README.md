# 🍽️ Good Host

Sistema visual para gerenciamento de mesas em restaurantes, com organização por seções e suporte a drag and drop.

---

## 🚀 Funcionalidades

* Organização de mesas por seção, divisão e linha
* Reordenação de mesas via drag and drop
* Persistência local com localStorage

---

## 🛠️ Tecnologias

* Angular 19
* Angular CDK (Drag & Drop)
* Angular Material
* TypeScript

---

## ▶️ Como rodar

```bash
npm install
ng serve
```

Acesse: http://localhost:4200
